Tipos de datos

string -> cadena de texto

Interger -> números enteros

Float -> números con decimales

Boolean -> Verdadero o Falso

### Data types

### String
~~~
print(type("Hello World"))
print("Hello"[0])
print("123"+"456")
~~~
### int
~~~
print(type(123))
print(123 + 45)
~~~
### float
~~~
print(type(123.456))
print(123.456 + 45.789)
~~~
### Boolean
~~~
print(type(True))
False
True
~~~
### Casteo    
~~~
print(type(int("123")))
print(int("123"))
print(type(float("123.456")))
print(float("123.456"))
print(type(str(123)))
print(str(123))
~~~
### F-strings
### Data manipulation
~~~
print(f"{123}")
print(f"{123.456}")
print(f"{'Hello'}")
~~~
